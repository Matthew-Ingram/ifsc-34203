Welcome

This site is intended to act as a form of documentation for various features of the HTML and CSS languages. HTML and CSS together act as two of the fundamental building blocks for building websites. As with many computer-related languages, having a good knowledge base for the features offered in HTML and CSS helps with planning and problem solving for a given project. Additionally, understanding modern approaches and standards for using HTML and CSS will help with keeping websites flexible, responsive, and searchable.
